## Requirements

| Requirement | Specification |
|-------------|---------------|
| **Minecraft Version** | 1.21.5+ |
| **Server Software** | Paper, Folia, Purpur or compatible forks |
| **Java Version** | Java 25+ |

## Installation

1. **Stop your server** completely
2. Download the latest `.jar` file from any source above
3. Place it in your server's `plugins/` folder
4. **Start your server** (do not use `/reload`)

See the [Installation Guide](/docs/installation) for full setup instructions.
